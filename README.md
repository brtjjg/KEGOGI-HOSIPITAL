# 🏥 Kegogi Medicare Hospital Website

A modern, responsive hospital website for **Kegogi Medicare Hospital** – a fully functional static site with an admin dashboard, appointment booking, and service showcase.

## ✨ Features

- **Hospital landing page** with hero, services, doctor profiles, testimonials, and contact.
- **Appointment & contact forms** with client‑side validation and success toasts.
- **Admin dashboard** (toggle via the “Admin” button) showing statistics, recent appointments, and patient lists.
- **Fully responsive** – works on desktop, tablet, and mobile.
- **Accreditation badge** (SIYA Social Health Authority) and 24/7 service highlight.
